# 💻 Mac Geçiş Rehberi

## HIZLI BAŞLANGIÇ

### Mac'te Terminal'i Açın
Command + Space → "Terminal" yazın

### Projeyi Klonlayın
```bash
cd ~/Documents
git clone https://github.com/toftamars/satiss-dashboard.git
cd satiss-dashboard
```

### Server Başlatın
```bash
python3 -m http.server 8000
```

### Tarayıcıda Açın
http://localhost:8000

## GÜNCELLEMautorELER

### Windows'tan Mac'e Çekme
```bash
cd ~/Documents/satiss-dashboard
git pull origin main
```

### Mac'ten Windows'a Gönderme
```bash
git add .
git commit -m "Mac'ten değişiklik"
git push origin main
```

## YAYGN KOMUTLAR

- Terminal Aç: Command + Space → "Terminal"
- VS Code Aç: code .
- Finder Aç: open .
- Server Durdur: Control + C

## GİT AUTH
İlk push için:
https://github.com/settings/tokens
→ Token oluştur
→ Push yaparken token'ı gir
